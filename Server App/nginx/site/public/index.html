Mac Client Install

For macOS Apple Silicon clients. Install the Pummelchen Swift client app from the DMG. The app handles sync status, updates, release history, shader/resource-pack defaults, and client repair from its GUI.

Download Mac Installer DMG Latest version: 2026-06-13_V23

Add A Mod

Send a plain operator request through Codex, OpenCode, Qwen, or another AI coding tool with access to this project and the VPS.

"Add this mod: https://www.curseforge.com/minecraft/mc-mods/example-mod"

The Swift server app resolves the mod, downloads the correct NeoForge file, classifies whether it belongs on the server, client, or both, updates DuckDB, rebuilds release artifacts, runs the configured validation hooks, and only promotes the release when the release pipeline accepts all required checks.

Do not copy mod files into the live folders by hand. Manual installs skip dependency records, validation, release manifests, client sync metadata, and rollback history.

Safe World Reset

Send the request through Codex, OpenCode, Qwen, or another AI coding tool as an operator command, for example:

"Make a Safe World Reset with a new random seed"

"Make a Safe World Reset with seed 12345"

The Swift server app plans the reset, requires destructive confirmation, removes the old active world safely, applies the configured seed, reinstalls required datapacks, reapplies safety gamerules, starts the server, pregenerates a 1000-block radius around spawn, verifies completion, and writes the result to DuckDB.

Do not delete world folders manually. Manual deletion skips Swift validation, audit records, datapacks, gamerules, backups, and pregeneration.
