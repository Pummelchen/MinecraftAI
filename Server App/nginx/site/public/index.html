Safe World Reset

Send the request through Codex as an operator command, for example:

"Make a Safe World Reset with a new random seed"

"Make a Safe World Reset with seed 12345"

The Swift server app plans the reset, requires destructive confirmation, removes the old active world safely, applies the configured seed, reinstalls required datapacks, reapplies safety gamerules, starts the server, pregenerates a 1000-block radius around spawn, verifies completion, and writes the result to DuckDB.

Do not delete world folders manually. Manual deletion skips Swift validation, audit records, datapacks, gamerules, backups, and pregeneration.
